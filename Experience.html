Detection of Counterfeit Products using Convolutional Neural NetworkDetection of Counterfeit Products using Convolutional Neural Network
• Developed a Convolutional Neural Network (CNN)-based image detection model using TensorFlow and Keras, achieving an impressive 79% validation accuracy and 74% testing accuracy.
• Scraped over 10,000 diverse images from Reddit using Selenium and Beautiful Soup, effectively expanding the dataset and significantly enhancing the model's performance.
• Employed advanced transfer learning techniques with Sentence-BERT (SBERT) for efficient vectorization, resulting in a remarkable 15% reduction in training time and a notable improvement in classification accuracy.
• Innovatively designed and implemented a user-friendly web application using Flask, creating a seamless and intuitive interaction platform for users to engage with the detection model, ensuring a smooth user experience.
